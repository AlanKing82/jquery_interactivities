Waste hierarchy in action

We must follow the waste hierarchy when we are using resources.

  • Recycle. (You should only consider this option when prevention and reuse have been ruled out). If you can’t recycle, then….
  • Consider if the item can be reused. If you can’t reuse then….
  • Dispose responsibly. If none of the options higher up in the waste hierarchy are available, only then may this option be used.
  • Prevent waste wherever possible. If you can’t prevent, then….
  • Recover a value (eg energy). If you can’t recover, then….
